# 🍇 Império Açaí Ceilândia — Site Oficial

Site institucional e de vendas do **Império Açaí**, localizado em Ceilândia, Distrito Federal. Desenvolvido para divulgar o cardápio, facilitar pedidos via WhatsApp e apresentar a localização do estabelecimento.

---

## 📁 Estrutura do Projeto

```
SITE-IMPERIO-ACAI/
├── index.html        # Página principal
├── style.css         # Estilos personalizados
├── README.md         # Documentação do projeto
└── img/
    ├── acai-foto.jpg  # Imagem de fundo do hero
    ├── logo-acai.png  # Logotipo (favicon)
    └── produto1.jpg   # Foto dos produtos no cardápio
```

---

## 🚀 Funcionalidades

- **Hero Section** com chamada para ação e botão de pedido via WhatsApp
- **Cardápio** com os 4 tamanhos disponíveis (300ml, 500ml, 700ml e 1L)
- **Avaliações do Google** integradas via Elfsight
- **Mapa do Google Maps** com a localização exata da loja
- **Links diretos para WhatsApp** em todos os botões de pedido

## 📦 Como Usar

1. Clone ou baixe o repositório
2. Adicione as imagens na pasta `img/`:
   - `acai-foto.jpg` — foto de fundo do hero
   - `logo-acai.png` — logotipo da marca
   - `produto1.jpg` — foto dos produtos
3. Abra o `index.html` em qualquer navegador

> Não requer instalação de dependências. Todas as bibliotecas são carregadas via CDN.
